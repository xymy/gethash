Gethash is a command-line hash utility.

Getting Started
---------------

Install from PyPI:

.. code-block:: shell

    $ pip install gethash

.. note::

    Gethash requires Python 3.7 and later.

Show command-line usage:

.. code-block:: shell

    $ gethash --help
